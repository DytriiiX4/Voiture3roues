# Voiture3roues

Voiture télécommandée à trois roues. Elle supporte la caméra dont le flux vidéo sera analysé par l'IA finale.

Le répertoire Joystick contient le code permettant d'envoyer les informations émisent par un joystick vers la voiture en Bluetooth. Le répertoire Pont_en_H contient le code permettant de piloter la voiture à partir de ces informations.
